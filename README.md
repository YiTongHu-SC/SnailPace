# 《蜗速前进》 Snail's-Pace

> 使用Unity开发的游戏Demo

## 游戏介绍

> GameJam游戏作品，一款 PC 端的卡牌+竞速游戏。玩家作为一只蜗牛，不断前进，与途中的敌人进行战斗，直到抵达终点

- 蜗速前进（Snail's Pace）| 2022.10-2022.11
- 平台：PC
- 玩法：玩家每回合最多3张手牌，打出3张手牌，或者主动刷新手牌。抵达终点即为胜利，通过分数排行榜可以看到自己的分数排名
- 特色内容
  - 卡牌+竞速
  - 速度排行榜
 
## 开发
